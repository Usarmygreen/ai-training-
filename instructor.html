🧠 What Is AI Actually Doing?
Plain language explanation of next-word prediction and hallucination patterns.
🎙️ Facilitator Script

"AI doesn't 'think' or understand facts. It is trained to predict the next most probable word based on massive patterns. This means AI will generate syntactically perfect code or IP addresses that are completely fabricated — a phenomenon called 'hallucinating.' This is why we verify, always."

  • Ask: "Has anyone run into an AI hallucination yet? What did it make up?"
  • Point to the Fluency Cheat Sheet on the Level 1 page for their quick reference.

AI Policy Breakdown (Critical)
Review the two-tier usage rule. Explain data boundaries and public sharing risk.
🎙️ Facilitator Script

"We enforce a strict two-tier policy: New L1 employees use Atlassian Rovo only for work tasks. Rovo is enterprise-governed, data stays internal, and it is native to Jira. Senior L1 and L2+ staff may use external AIs (like Claude or Gemini) but must manually strip PII, credentials, or proprietary source code before pasting. You are fully responsible for what you deploy."

  • Walk through the Enterprise vs. Consumer comparison table on Level 2 page.

12:20 PM – 12:32 PM (12 mins)
Group Activity 1: PowerShell Roulette
Groups audit an AI script disabling AD users. Run diagnostics in activities panel.
🎙️ Facilitator Script & Answer Key

"Let's open our first group exercise: PowerShell Roulette. We asked the AI to disable stale AD accounts. Split into groups of 3 and take 5 minutes to audit the AI's script. What errors can you find?"

Answer Key — 3 Critical Bugs:

  • Query Filter Failure: The AD Active Directory module cannot evaluate LastLogonDate inside the -Filter query. The cmdlet throws a fatal parser exception.
  • Safety Flag Missing: There is no -WhatIf flag. Running this would immediately write state changes to the active domain controllers, risking accidental system lockout.
  • Deprecated cmdlet: Send-MailMessage is officially deprecated and insecure (lacks modern TLS).
12:32 PM – 12:40 PM (8 mins)
✍️ Prompt Engineering & PCTF
Introduce the PCTF framework. Demonstrate the interactive Prompt Composer.
🎙️ Facilitator Script

"Vague prompts yield generic, hallucination-prone results. Using the **PCTF framework** (Persona, Context, Task, Format) grounds the model. Let's look at the **Interactive Prompt Playground** on the homepage to demonstrate how we structure these fields for system admins."

  • Show them how tweaking the 'Format' dropdown dynamically forces the output structure.
  • Emphasize utilizing 'Custom Details' to input specific OS versions (e.g. Windows 11 vs Windows 7).
12:40 PM – 12:52 PM (12 mins)
Group Activity 2: Apple Silicon SMC Trap
Pairs run the macOS M2 troubleshoot diagnostic. Highlight SMC platform errors.
🎙️ Facilitator Script & Answer Key

"Let's look at Activity 2: The Apple Silicon platform trap. The prompt describes a Bluetooth issue on an M2 Mac. AI suggests holding Shift-Control-Option-Power to reset the SMC chip."

Answer Key — SMC does not exist:

  • SMC (System Management Controller) was an Intel-based chip. Apple M-chips manage these functions directly. The instructions waste user time and diminish service desk credibility.
  • Have them click "Run Intel SMC command" to show the failure. Show the "Apple Silicon Bluetooth Fix" terminal to illustrate configuration cleanups.
