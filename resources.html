Microsoft Documentation — Verification Resources
Always verify AI-generated Windows and PowerShell content against official Microsoft docs before applying

Quick Policy Reference
The rules — at a glance
New L1 Employees: Use Atlassian Rovo only for any work-related AI tasks. Do not use ChatGPT, Claude.ai (free), or any consumer AI tool for work. When in doubt, ask your team lead.
Senior L1 / L2+ Employees: You may use any AI tool, but you must vet every output before use. Never input PII, client data, proprietary code, internal financials, passwords, or trade secrets into consumer AI. Document AI contributions to deliverables.
Personal Use Accelerator: We highly encourage using AI tools (ChatGPT, Claude, Gemini) in your daily life outside the office. Planning events, drafting messages, summarizing books, or working on personal code builds your "prompting muscle memory" and tool familiarity. This personal fluency directly translates to 10x higher confidence and efficiency when using Atlassian Rovo at work.
Universal Rule: AI is a tool, not an oracle. You are accountable for every output you use or share. Read it, check it, verify it — then use it.
